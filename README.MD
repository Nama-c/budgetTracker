###############################
        BudgetTracker
###############################

# DESCRIPTION:
    Une application web et console qui permet de suivre ses dépenses et de gérer son budget personnel. Elle offre des fonctionnalités telles que l'ajout de transactions, la visualisation des dépenses par catégorie.

# Stack:
    ## WEB:
        - Frontend: HTML, CSS, Tailwind CSS
        - Backend: PHP
        - Database: PostgreSQL
    ## CONSOLE:
        - Language: Java
        - Database: PostgreSQL
    ## ANALYSE:
        - Modelisation: UML(StarUML)

# Users:
    - Administrateur
    - Utilisateur

# Features:
    ## Administrateur:
        - Gestion des utilisateurs
        - Gestion des catégories de dépenses
        - Visualisation des statistiques globales
    ## Utilisateur:
        - Ajout de transactions (dépenses et revenus)
        - Visualisation des dépenses par catégorie
        - Gestion du budget personnel
